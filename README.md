# Data-Visualization
Superstore Dashboard – Task 2

 Overview
This project is part of my Data Analyst Internship Task 2.  
It focuses on data visualization and storytelling using Power BI.

Files in Repository
- 'Sample - Superstore.csv' → Original dataset
- 'cleaned_superstore.xlsx' → Cleaned dataset
- 'dashboard.pbix' → Power BI dashboard file
- '/screenshots/'→ Dashboard screenshots
- 'README.md' → Project documentation

 Key Insights
- Technology category drives consistent growth.
- Furniture sales are high but profit margins are low.
- Discounts negatively impact profit.

 Tools Used
- Power BI
- Excel (for cleaning)
- GitHub (for submission)
